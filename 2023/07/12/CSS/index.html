平面转换

平移

属性： transform: translate(X轴移动距离,Y轴移动距离);

取值：

  • 像素px
  • 百分比（参照盒子自身尺寸计算结果）
  • 正负均可
  • 只写一个数，表示水平方向

旋转

属性： transform: rotate(旋转角度);

取值：

  • 角度单位为 deg
  • 正值为顺时针旋转，负值为逆时针旋转
改变转换原点

属性：transform-origin: 水平原点位置 垂直原点位置;

取值：

  • 方位名词（left、top、right、bottom、center）
  • 像素px
  • 百分比

多重转换技巧

先平移再旋转，因为多重转换以第一种转换形态的坐标轴为准，旋转会改变坐标轴向

缩放

属性： transform: scale(缩放倍数) transform: scale(X轴缩放倍数，Y轴缩放倍数)

取值：

  • 设置一个值就是X轴和Y轴等比例缩放
  • 取值大于1为放大，取值小于1为缩小

倾斜

属性： transform: skew();

取值：角度度数 deg

渐变

线性渐变

属性：

取值：

  • 渐变方向（可选）
    • to 方位名词
    • 角度度数
  • 终边位置（可选）：百分比

径向渐变

属性：

取值：

  • 半径是2条时为椭圆
  • 圆心位置2条：像素px/百分比/方位名词

空间转换

平移

transform3d: (x,y,z)

视距

作用：指定了观察者与z=0平面的距离，为元素添加透视效果，这样就能观察到空间转换时z轴的变化

透视效果：近大远小，近实远虚

属性：（添加给直接父级，取值范围800-1200）

perspective: 视距;

旋转

transform: rotateX(角度);

transform: rotateY(角度);

transform: rotateZ(角度);

取值正负判断：左手定则

自定义旋转轴的位置及旋转角度： rotate3d(x,y,z,角度) ，xyz取值为0-1

立体呈现

作用：设置元素的子元素位于平面或3D空间中

属性名： transform-style

属性值：

  • flat：子级处于平面中
  • preserve-3d：子级处于3D空间

呈现立体图形步骤：

  • 父元素添加 transform-style: preserve-3d;
  • 子级定位
  • 调整子盒子位置（位移或旋转）

缩放

transform: scale3d(x,y,z);
