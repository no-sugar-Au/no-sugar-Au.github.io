本机端口占用问题

Posted on 2023-02-27 In 学习发现

查看本机端口占用情况

起因：在运行使用 npm run dev 命令运行webpack项目时，发现本机8080端口被占用

解决方法：在cmd命令行中输入 netstat -ano 可以查看本机端口占用情况

如果发现自己某个需要使用的端口被占用，可以在任务管理器->详细信息中找到相对应的PID，直接关闭
